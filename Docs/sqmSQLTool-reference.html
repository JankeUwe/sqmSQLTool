🚀 Quick Start

sqmSQLTool is a modular PowerShell toolset for SQL Server administration with 128 functions zur Verwaltung von AlwaysOn, Backup/Restore, Performance, Sicherheit und mehr.
1
Import module
Import-Module sqmSQLTool

Set configuration
Set-sqmConfig -LogPath "C:\System\WinSrvLog\MSSQL" -OutputPath "C:\System\WinSrvLog\MSSQL"
3
Check instance
Get-sqmSQLInstanceCheck -SqlInstance "SQL01"
4
AlwaysOn Status
Get-sqmAlwaysOnHealthReport -SqlInstance "SQL01"
5
Setup Report
Invoke-sqmSetupReport -SqlInstance "SQL01"

# Without internet access (z.B. Bankenumgebung):
# Provide dbatools as ZIP auf einen UNC share and und extract manually to:
# C:\Program Files\WindowsPowerShell\Modules\dbatools\

Download sqmSQLTool
# From GitHub as ZIP:
# https://github.com/JankeUwe/sqmSQLTool
# → Code → Download ZIP → extract

Installation with Install.cmd (recommended)

Install.cmd automatically detects whether admin rights vorhanden sind, handles the execution policy und installiert ins richtige PowerShell-Verzeichnis.

Install.cmd                  # Auto: AllUsers if admin, sonst CurrentUser
Install.cmd AllUsers         # System-wide (UAC prompt if needed)
Install.cmd CurrentUser      # Nur für aktuellen Benutzer, keine Elevation nötig
